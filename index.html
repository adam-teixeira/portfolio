Objetivo

Sou um homem trans de 23 anos que encontra inspiração em livros, pincéis, videogames e no aprendizado de novos idiomas, participando até mesmo do Comitê Nacional Preparatório do WYF como voluntário. Meu objetivo é usar a tecnologia como ponte, tornando-a mais empática, inclusiva e acessível.

Experiência

Atuo como estagiário apoiando alunos da trilha de Programação FullStack e participo como voluntário de ações do Comitê de Diversidade e Inclusão no Instituto Atlântico. Como implementar o CTE em PL\SQL

Sigla para Common Table Expression (expressão de tabela comum), A CTE é uma tabela temporária que você pode criar dentro de uma consulta SQL e usar como se fosse uma tabela normal. É uma alternativa às subconsultas (consultas aninhadas dentro de outras consultas).

Como simplificar consultas complexas em Banco de Dados: CTE.

Montar uma consulta SQL é uma tarefa que exige muita atenção, especialmente quando é preciso juntar várias demandas em um só código. Quando você se depara com aquela enchurrada de códigos robustos, consultas dentro de consultas, é comum que o desenvolvedor se sinta perdido e não saiba por onde começar. É nesses casos que podemos usar a CTE!
